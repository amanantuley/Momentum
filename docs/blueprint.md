# **App Name**: Task Tracker

## Core Features:

- Task Input: Allow users to input and name their daily tasks.
- Daily Tracking: Display tasks in a calendar format where users can mark them as complete for each day.
- Progress Summary: Provide a monthly and yearly summary of task completion rates.

## Style Guidelines:

- Primary color: Calm light blue (#ADD8E6) to evoke a sense of serenity and focus.
- Background color: Very light gray (#F5F5F5) for a clean and non-distracting interface.
- Accent color: Soft green (#90EE90) to indicate completed tasks, providing a positive reinforcement.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif for a blend of readability and contemporary style.
- Simple and intuitive icons for task categories to enhance visual clarity.
- Clean and grid-based layout to mirror the original spreadsheet design.
- Gentle fade-in and checkmark animations when tasks are marked complete to provide subtle feedback.
